# JavaScript DBX
## A simple mockup of a database in nodejs.

> Supports indexing on any field(s) (dynamically maintained)  
> Supports queries on multiple fields:  
  - Find One  
  - Find Many  
  -- Match on all fields  
  -- Match on any field  

### API:
:: load(callback)  
:: save()  
:::: createIndex(key)  
:::: removeIndex(key)  
:: insert(data)  
:::: find(query)  
:::: findOne(query)  
:::: findAny(query)  
:::: findAnyOne(query)  
:: update(query, data)  
:: updateOne(query, data)  
:: updateAny(query, data)  
:: updateAnyOne(query, data)  
:::: remove(query)  
:::: removeOne(query)  
:::: removeAny(query)  
:::: removeAnyOne(query)  
:: count()
